# Spices
Save recipes from any website and automatically clean them into a simple, readable format by removing ads, stories, and clutter for a distraction free cooking experience.


What

This is a recipe saver web app that lets users paste a recipe URL and instantly view a clean, readable version of the recipe. It extracts key information like ingredients and instructions while removing ads, popups, and clutter.

Why

Most recipe websites are difficult to use because they are filled with ads, long personal stories, and messy formatting before you ever reach the actual recipe. Saving recipes also often requires accounts, paywalls, or clunky interfaces. This project focuses on making recipes easy to access, read, and use without distractions.

How

Users can paste a recipe link into the app. The system extracts the recipe content, processes it into structured sections (title, ingredients, and steps), and displays it in a clean format. The goal is a simple workflow: find a recipe online → paste link → instantly get a usable cooking view.
